Plot Variables for Subgroups Against the Entire Sample

This procedure produces small charts for selected variables across subgroups of the data. Each chart displays the distribution for the entire sample with the distribution for the subgroup overlaid on it. This may be particularly useful for understanding clustering results, but any categorical variable can be used to define the groups.

Subgroups Defined by Select a variable to define the subgroups to be plotted. Charts will be generated for each distinct value of the variable.

Variables to Plot Select the variables whose distribution will be plotted. For categorical variables, bar charts are produced; for scale variables, the charts are histograms. If the number of variables exceeds the row size limit (see the Options subdialog), each set of charts will take multiple rows.

• Graphs do not include tick marks or labels due to space constraints. For scale variables, the axis range is from the data minimum to the data maximum for the entire dataset. For categorical variables, all labelled values and any other values found in the subgroup are included.

Cases are sorted by the subgroup variable Check this box if the data are already sorted by the subgroup variable in order to save an extra sorting step.

Options

Individual Chart Size Enter the horizontal and vertical dimensions of the chart in inches. This size refers to the individual charts; not the overall size of the display. If the horizontal size is two inches, and there are five variables, the set of charts for each subgroup would be ten inches wide.

Vertical Chart Scaling Percentage Enter a percentage to shrink the chart data area by in the vertical dimension. The default value, 90, need only be changed if the horizontal axis labels are clipped, which can happen with very small charts.

Charts Per Row By default, all the charts for a subgroup are displayed in one row (up to ten). Specify the maximum number of charts to appear in one row. Additional rows of charts for each subgroup are displayed as needed.

Transparency Percentage Each chart shows the distribution of the variable for the subgroup overlaid on the distribution in the entire dataset. The transparency percentage controls how the entire distribution shows through the subgroup distribution. 0 means no transparency, and 100 means complete transparency.

Histogram Shape Choose Area to display the histograms as areas and Bars to display them with bars. Choose Kernel Smoother to display a smoothed density function.

• With Kernel Smoother, the entire-group and subgroup distributions are each scaled to the number of points in them. For the others, the two charts are on the same scale.

Proportion for Kernel Smoother specifies the proportion of points that are used in calculating the density function estimate.

Missing Values Choose the missing value treatment. If By variable, each graph uses all the data available on the variable. If Listwise, only the cases where none of the graphed variables is missing are used in each graph. By variable maximizes the data in each graph; Listwise ensures a consistent case base across graphs.

Outline Label Enter text to be used to label each set of graphs in the Viewer outline. The subgroup identification is appended to this text.

Temporary Directory A temporary directory can be selected for the intermediate data file created by this procedure. This value only needs to be set if the procedure cannot otherwise find a temporary directory. For technical reasons, this procecdure cannot delete the file that it writes to this directory. Specifying a specific directory would make it easier to find and delete such files.

Colors Select colors for the entire-dataset bars and the subgroup bars. A selection of colors is listed here, but many more are available in syntax. See the main help under GPL Color Constants for a complete list.

Patterns Select patterns for the entire-dataset and subgroup bars. By default, bars are solid.

Additional Features

This dialog generates syntax for the STATS SUBGROUP PLOTS extension command. To display syntax help for this command, run the following syntax:

STATS SUBGROUP PLOTS /HELP.

Requirements

This command requires the Python Essentials and the STATS SUBGROUP PLOTS extension command and at least version 18 of IBM SPSS Statistics. You can download these from the SPSS Community website at www.ibm.com/developerworks/spssdevcentral.
