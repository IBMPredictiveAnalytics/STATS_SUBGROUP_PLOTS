Tracer les variables sous forme graphique pour les sous-groupes dans l'échantillon entier

Cette procédure produit de petits graphiques pour les variables sélectionnées dans des sous-groupes de données. Chaque graphique affiche la distribution de l'échantillon entier avec la distribution du sous-groupe par dessus. Cela peut être particulièrement utile pour comprendre les résultats de classification mais n'importe quelle variable catégorielle peut être utilisée pour définir les groupes.

Sous-groupes définis par Sélectionnez une variable pour définir les sous-groupes à tracer. Des graphiques seront générés pour chaque valeur distincte de la variable.

Variables à tracer Sélectionnez les variables dont la distribution sera tracée. Pour les variables catégorielles, des graphiques à barres sont produits ; pour les variables d'échelle, les graphiques sont des histogrammes. Si le nombre de variables dépasse la limite de taille de ligne (voir la sous-boîte de dialogue Options), chaque ensemble de graphiques utilisera plusieurs lignes.

• Les graphiques ne contiennent pas de coches ou de libellés en raison d'un espace limité. Pour les variables d'échelle, l'axe s'étend des valeurs de données minimum aux valeurs de données maximum pour tout le jeu de données. Pour les variables catégorielles, toutes les valeurs étiquetées et les autres valeurs trouvées dans le sous-groupe sont incluses.

Les observations sont triées selon la variable de sous-groupe Cochez cette case si les données sont déjà triées selon la variable de sous-groupe afin d'économiser une étape de tri supplémentaire.

Options

Taille de graphiques individuels Saisissez les dimensions horizontale et verticale du graphique en pouces. Cette taille concerne les graphiques individuels et non la taille générale de l'affichage. Si la taille horizontale est de deux pouces et qu'il y a cinq variables, l'ensemble de graphiques de chaque sous-groupe sera d'une largeur de 10 pouces.

Pourcentage de mise à l'échelle verticale du graphique Saisissez un pourcentage pour réduire la zone des données du graphique dans la dimension verticale. La valeur par défaut, 90, ne doit être modifiée que si les libellés de l'axe horizontal sont tronqués, ce qui peut arriver avec les très petits graphiques.

Graphiques par ligne Par défaut, tous les graphiques d'un sous-groupe sont affichés sur une ligne (jusqu'à dix). Spécifiez le nombre maximal de graphiques à afficher sur une ligne. Des lignes de graphiques supplémentaires pour chaque sous-groupe sont affichées en fonction des besoins.

Pourcentage de transparence Chaque graphique affiche la distribution de la variable pour le sous-groupe par dessus la distribution dans le jeu de données entier. Le pourcentage de transparence contrôle la façon dont la distribution entière apparaît sur la distribution du sous-groupe. 0 signifie aucune transparence et 100 signifie une transparence totale.

Forme de l'histogramme Choisissez Zone pour afficher les histogrammes sous la forme de zones et Barres pour les afficher sous la forme de barres. Choisissez Lisseur de noyau pour afficher une fonction de densité lissée.

• Avec le Lisseur de noyau, les distributions entières des groupes et des sous-groupes sont mises à l'échelle selon le nombre de points qu'ils contiennent. Pour les autres, les deux graphiques se trouvent sur la même échelle.

La Proportion du lisseur de noyau spécifie la proportion de points qui sont utilisés pour calculer l'estimation de la fonction de densité.

Valeurs manquantes Choisissez le traitement des valeurs manquantes. S'il s'agit de l'option Par variable, chaque graphique utilise toutes les données disponibles sur la variable. S'il s'agit de l'option Toute observation incomplète, seules les observations dans lesquelles aucune variable représentée graphiquement ne manque sont utilisées dans chaque graphique. Par variable augmente le nombre maximal de données dans chaque graphique. Valeur incomplète garantit une base d'observations cohérente dans les graphiques.

Libellé de légende Saisissez le texte à utiliser pour étiqueter chaque ensemble de graphiques dans la légende du visualiseur. L'identification de sous-groupe est ajoutée à ce texte.

Répertoire temporaire Un répertoire temporaire peut être sélectionné pour le fichier de données intermédiaire créé par cette procédure. Cette valeur n'est nécessaire que si la procédure ne trouve pas de répertoire temporaire. Pour des raisons techniques, cette procédure ne peut pas effacer le fichier qu'elle crée dans ce répertoire. Spécifier un répertoire spécifique facilite la recherche et la suppression de ce genre de fichiers.

Couleurs Sélectionnez les couleurs des barres de tout le jeu de données et des sous-groupes. Une sélection de couleurs est répertoriée ici mais d'autres couleurs sont disponibles dans la syntaxe. Consultez l'aide principale sous Constantes de couleur GPL pour obtenir une liste complète.

Motifs Sélectionnez des motifs pour les barres de tout le jeu de données et des sous-groupes. Par défaut, les barres sont de couleur unie.

Fonctions supplémentaires

Cette boîte de dialogue génère une syntaxe pour la commande d'extension STATS SUBGROUP PLOTS. Pour afficher l'aide sur la syntaxe de cette commande, exécutez la syntaxe suivante :

STATS SUBGROUP PLOTS /HELP.

Conditions requises

Cette commande nécessite Python Essentials et la commande d'extension STATS SUBGROUP PLOTS et au moins la version 18 d'IBM SPSS Statistics. Vous pouvez les télécharger depuis le site Web de la communauté SPSS à l'adresse www.ibm.com/developerworks/spssdevcentral.
