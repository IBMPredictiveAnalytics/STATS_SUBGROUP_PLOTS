Графики переменных для подгрупп в сравнении с графиком для всей выборки

Эта процедура строит небольшие диаграммы для выбранных переменных по всем подгруппам набора данных. На каждой диаграмме показано распределение для выборки в целом, на которое наложены частные распределения по подгруппам. Это может быть особенно полезно для понимания результатов кластеризации, хотя группы могут быть заданы с помощью любой категориальной переменной.

Подгруппы, заданные с помощью Выберите переменную, чтобы задать подгруппы для построения графика. Для каждого отдельного значения этой переменной будет сгенерирована диаграмма.

Переменные для построения графиков Выберите переменные, для которых нужно построить графики распределения. Для категориальных переменных строятся столбчатые диаграммы, для количественных переменных строятся гистограммы. Если число переменных превышает предельную длину ряда (смотрите диалоговое окно Опции), каждый набор диаграмм будет располагаться в несколько рядов.

• Из-за недостатка места на этих графиках не показаны отметки делений и метки. В случае количественных переменных диапазон оси отсчитывается от минимального до максимального значения для всего набора данных. Для категориальных переменных используются все значения с метками и все остальные значения, найденные в подгруппе.

Наблюдения отсортированы по переменной подгруппы Включите этот переключатель, если данные уже отсортированы по переменной подгруппы, чтобы избежать лишнего шага сортировки.

Параметры

Размер отдельной диаграммы Введите горизонтальный и вертикальный размер диаграммы в дюймах. Этот размер относится к отдельным диаграммам, а не к общему размеру области вывода. Если горизонтальный размер составляет два дюйма и используется пять переменных, то ширина набора диаграмм для каждой подгруппы составит десять дюймов.

Процент шкалирования диаграммы по вертикали Введите процент сжатия области данных диаграммы в вертикальном измерении. Значение по умолчанию (90) нужно менять только в случае, если метки горизонтальной оси оказываются обрезанными, что может произойти только на очень маленьких диаграммах.

Диаграмм в ряду По умолчанию все диаграммы для подгруппы выводятся в одном ряду (до десяти). Задайте максимальное количество диаграмм, выводимых в одном ряду. При необходимости выводятся дополнительные ряды диаграмм для каждой подгруппы.

Процент прозрачности На каждой диаграмме показано распределение переменной для подгруппы, наложенное на распределение для всего набора данных. Процент прозрачности задает, насколько хорошо полное распределение видно сквозь распределение для подгруппы. 0 означает отсутствие прозрачности, а 100 - полную прозрачность.

Форма гистограммы Выберите Область, чтобы показывать гистограммы в виде областей, и Столбцы, чтобы показывать их в виде столбцов. Выберите Ядерное сглаживание для вывода сглаженной функции плотности.

• При ядерном сглаживании распределения для всей группы и для подгрупп шкалируются по количеству точек в них. При других способах обе диаграммы имеют одинаковую шкалу.

Пропорция для ядерного сглаживания задает долю точек, используемых в расчетах при оценке функции плотности.

Пропущенные значения Выберите способ обработки пропущенных значений. Если выбрано По переменным, для каждого графика будут использованы все доступные данные по этой переменной. Если выбрано Целиком, будут использованы только те наблюдения, для которых не пропущено ни одного значения переменных, показанных на графике. Опция По переменным максимизирует длину рядов данных для каждого графика, опция Целиком обеспечивает единую структуру данных для каждого графика.

Метки схемы Введите текст, который будет служить меткой для каждого набора графиков в средстве просмотра. К этому тексту добавлена идентификация подгрупп.

Временный каталог Можно выбрать временный каталог для файла промежуточных данных, созданных этой процедурой. Это значение нужно задать, только если процедура не может найти временный каталог другим способом. По техническим причинам эта процедура не может удалить файл, который она записывает в этот каталог. Создание специального каталога облегчает поиск и удаление таких файлов.

Цвета Выберите цвета для столбцов, относящихся ко всему набору данных и к данным подгрупп. Цвета для выбора перечислены здесь, но большое количество цветов можно также задать через синтаксис. Полный список смотрите в главной справке под заголовком Цветовые константы GPL.

Узоры Выберите узоры для столбцов всего набора данных и столбцов подгрупп. По умолчанию столбцы имеют сплошную заливку.

Дополнительные возможности

Это диалоговое окно генерирует синтаксис для команды расширения STATS SUBGROUP PLOTS. Чтобы показать справку синтаксиса, выполните следующий синтаксис:

STATS SUBGROUP PLOTS /HELP.

Технические требования

Для выполнения этой команды требуется Python Essentials, команда расширения STATS SUBGROUP PLOTS и IBM SPSS Statistics версии 18 или новее. Можно скачать их с сайта SPSS Community по адресу www.ibm.com/developerworks/spssdevcentral.
