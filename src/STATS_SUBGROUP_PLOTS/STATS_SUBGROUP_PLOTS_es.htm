Representar variables para subgrupos frente a toda la muestra

Este procedimiento genera gráficos pequeños para variables seleccionadas en subgrupos de datos. Cada gráfico muestra la distribución de toda la muestra con la distribución del subgrupo superpuesta. Esto puede ser particularmente útil para comprender los resultados de la agrupación en clústeres, pero se puede utilizar cualquier variable categórica para definir los grupos.

Subgrupos definidos por Seleccione esta variable para definir los subgrupos que se van a representar. Se generarán gráficos para cada valor distinto de la variable.

Variables para representar Seleccione las variables cuya distribución se va a representar. Para variables categóricas, se generan gráficos de barras; para variables de escala, se generan histogramas. Si el número de variables supera el límite de tamaño de fila (vea el subcuadro de diálogo Opciones), cada conjunto de gráficos ocupará varias filas.

• Los gráficos no incluyen marcas de comprobación ni etiquetas por restricciones de espacio. En el caso de variables de escala, el rango de eje va desde los datos mínimos hasta los datos máximos de todo el conjunto de datos. En el caso de variables categóricas, se incluyen todos los valores con etiqueta y otros valores encontrados en el subgrupo.

Los casos se ordenan por la variable de subgrupo Marque esta casilla si los datos ya están ordenados por la variable de subgrupo para ahorrarse un paso de ordenación adicional.

Opciones

Tamaño de gráfico individual Introduzca las dimensiones horizontales y verticales del gráfico en pulgadas. Este tamaño hace referencia a los gráficos individuales; no al tamaño global de la visualización. Si el tamaño horizontal es de dos pulgadas, y hay cinco variables, el conjunto de los gráficos de cada subgrupo tendría diez pulgadas de ancho.

Porcentaje de escala de gráfico vertical Introduzca un porcentaje para reducir el área de los datos del gráfico en la dimensión vertical. El valor predeterminado, 90, sólo debe modificarse si las etiquetas de eje horizontal se muestran recortadas, que sucede en el caso de gráficos muy pequeños.

Gráficos por fila De manera predeterminada, todos los gráficos de un subgrupo se muestran en una fila (hasta diez). Especifique el número máximo de gráficos que desea que aparezcan en una fila. Se pueden mostrar filas adicionales de gráficos para cada subgrupo si así lo necesitara.

Porcentaje de transparencia Cada gráfico muestra la distribución de la variable del subgrupo superpuesto en la distribución del conjunto de datos completo. El porcentaje de transparencia controla cómo se muestra toda la distribución a través de la distribución del subgrupo. 0 significa sin transparencia, y 100 significa transparencia total.

Forma del histograma Elija Área para que los histogramas se muestren como áreas y Barras para que se muestren en forma de barras. Elija Suavizado de kernel para mostrar una función de densidad suavizada.

• Con Suavizado de kernel, las distribuciones de grupo completo y de subgrupo se escalan en el número de puntos que contienen. Para las demás distribuciones, los dos gráficos tienen la misma escala.

Proporción de suavizado de kernel especifica la proporción de puntos que se utiliza en el cálculo de la estimación de la función de densidad.

Valores perdidos Elija el tratamiento de los valores perdidos. Si es Por variable, cada gráfico utiliza todos los datos disponibles en la variable. Si es Por lista, en cada gráfico sólo se utilizan los casos en los que no falte ninguna de las variables dibujadas. Por variable maximiza los datos de cada gráfico; Por lista garantiza una base de casos coherentes en todos los gráficos.

Etiqueta de titulares Escriba el texto que se va a utilizar para etiquetar cada conjunto de gráficos en los titulares del visor. La identificación del subgrupo se añade a este texto.

Directorio temporal Se puede seleccionar un directorio temporal para el archivo de datos intermedio creado mediante este procedimiento. Este valor sólo debe establecerse si el procedimiento no puede encontrar un directorio temporal. Por razones técnicas, este procedimiento no puede suprimir el archivo que escribe en este directorio. La especificación de un directorio concreto facilitaría la búsqueda y supresión de dichos archivos.

Colores Seleccione los colores de las barras de todo el conjunto y de las barras del subgrupo. Aquí se lista una selección de colores, pero en la sintaxis dispone de muchas más opciones. Consulte la ayuda principal en el tema sobre las constantes de color GPL para obtener una lista completa.

Patrones Seleccione los patrones de las barras de subgrupo y del conjunto de datos completo. De manera predeterminadas, las barras son sólidas.

Características adicionales

Este cuadro de diálogo genera la sintaxis para el comando de extensión STATS SUBGROUP PLOTS. Para mostrar la ayuda de sintaxis de este comando, ejecute la siguiente sintaxis:

STATS SUBGROUP PLOTS /HELP.

Requisitos

Este comando requiere Python Essentials y el comando de extensión STATS SUBGROUP PLOTS y, como mínimo, la versión 18 de IBM SPSS Statistics. Puede descargarlos desde el sitio web de la Comunidad de SPSS en www.ibm.com/developerworks/spssdevcentral.
